# 📘 智能板书监控系统 v6.1 - 使用说明

## 一、快速开始

### 1.1 启动程序
- **直接运行**: 双击 `SmartBoardMonitor.exe`
- **从源码运行**: 执行 `python main.py`
- **静默启动**: 使用 `python main.py --silent` 参数

### 1.2 首次配置
1. 打开程序主界面（如已最小化，点击托盘图标）
2. 在"基本设置"标签页设置保存路径（建议使用非系统盘）
3. 设置截图间隔（默认60秒，可根据需要调整）

### 1.3 开始监控
- 点击主界面的"▶ 开始监控"按钮
- 状态栏显示"运行中"表示监控已启动
- 系统托盘图标变为运行状态

## 二、核心功能详解

### 2.1 固定间隔截图
- **定时截图**: 按设定时间间隔自动截图，无需用户活动检测
- **稳定可靠**: 不受用户输入影响，保证截图连续性
- **间隔可调**: 截图间隔从1秒到300秒可自由设置

### 2.2 智能去重功能
- **自动去重**: 使用MD5哈希算法自动识别重复截图
- **实时过滤**: 在缓冲区阶段即时过滤相同内容
- **节省空间**: 避免重复截图占用存储空间

### 2.3 进程监控
- **指定进程**: 在"高级设置"中输入进程名（如 `ppt.exe`, `notepad.exe`）
- **实时检测**: 程序会实时检查输入的进程是否存在
- **历史记录**: 自动保存使用过的进程名，方便下次选择

### 2.4 文件管理
- **自动分类**: 截图按 `年-月-日_星期` 格式分文件夹保存
- **规范命名**: 文件名为 `年-月-日-时-分-秒.png` 格式
- **计划保存**: 可设置每日固定时间自动保存截图

### 2.5 缓冲区管理
- **内存缓冲**: 截图先保存在内存中，避免频繁磁盘写入
- **智能保存**: 缓冲区满时自动保存到磁盘
- **体积预估**: 显示缓冲区预计占用的磁盘空间

## 三、高级功能

### 3.1 开机自启动
1. 在"系统设置"标签页勾选"开机自启动"
2. 首次设置可能需要管理员权限确认
3. 可同时开启"静默启动"（启动时不显示窗口）

### 3.2 实时设置应用
- **截图间隔**: 更改后立即生效，无需重启监控
- **窗口检测**: 切换前台窗口检测状态立即生效
- **进程设置**: 修改监控进程后立即生效

### 3.3 系统托盘
- **右键菜单**: 显示/隐藏窗口、开始/停止监控、退出程序
- **双击图标**: 切换窗口显示/隐藏状态
- **通知提示**: 重要操作会显示托盘通知

## 四、故障排除

### 4.1 常见问题

**Q: 开机自启后没有开始监控？**
A: 确保在"系统设置"中同时勾选"开机自启动"，程序会自动开始监控。

**Q: 截图没有保存？**
A: 检查保存路径是否有写入权限，确保磁盘空间充足。

**Q: 进程监控不工作？**
A: 检查进程名是否正确（需要 `.exe` 后缀），确认进程正在运行。

**Q: 程序卡顿或无响应？**
A: 尝试减少缓冲区大小，或增加截图间隔时间。

**Q: 去重功能如何工作？**
A: 程序使用MD5哈希算法计算截图特征值，相同内容的截图只保留一份。

## 五、注意事项

### 5.1 系统要求
- Windows 7/8/10/11 操作系统
- .NET Framework 4.5+（Windows 7可能需要安装）
- 建议4GB以上内存
- 足够的磁盘空间保存截图

### 5.2 隐私安全
- 本程序仅用于个人教学和工作记录
- 截图保存在本地，不上传任何数据
- 请勿用于监控他人计算机

### 5.3 性能建议
- 截图间隔建议不少于10秒
- 缓冲区大小建议50-200张截图
- 定期清理旧的截图文件
- 建议使用SSD硬盘提高保存速度
- 去重功能会自动运行，无需额外配置
